Welcome to your new blog! This example post shows you how to use all the features available when writing blog posts in Obsidian.

## Writing in Obsidian

You can write your blog posts in [Obsidian](https://obsidian.md), a beautiful markdown editor that makes writing a pleasure. Simply open the `src/content/blog/` folder as an Obsidian vault.

### Using Templates

When creating a new post, use the blog post template:
1. Create a new note (Cmd+N)
2. Open command palette (Cmd+P)
3. Search for "Templates: Insert template"
4. Select "Blog Post Template"

The template will automatically insert today's date and the frontmatter structure you need.

## Adding Images

Images work with **Approach B** - absolute paths. When you drag an image into Obsidian:

1. The image is saved to `public/assets/img/blog/`
2. Obsidian inserts: `![Description](/assets/img/blog/filename.png)`
3. The image appears broken in Obsidian (this is expected!)
4. The image appears perfectly in your browser when you run `npm run dev`

## Links

You can add links to [Reddit](https://reddit.com/r/ludicRPG), [YouTube](https://youtube.com/@ludicRPG), or any other site.

## Tags and Categories

Use tags in the frontmatter to categorize your posts:
- `alien-rpg` - Posts get green glow effect
- `cops-rpg` - Posts get blue glow effect
- Add any other tags you want!

## Reddit Discussion

After publishing your post, you can add a Reddit discussion link in the frontmatter:

```yaml
redditDiscussion: "https://reddit.com/r/ludicRPG/comments/xyz"
```

This will show a "Discuss on Reddit" button at the bottom of your post.

## Publishing Workflow

1. **Write** in Obsidian (`src/content/blog/`)
2. **Preview** in browser (`npm run dev` → `localhost:4321/blog`)
3. **Commit** with Git (or use Obsidian Git plugin for auto-commit)
4. **Push** to GitHub
5. **Deploy** happens automatically via Cloudflare Pages
